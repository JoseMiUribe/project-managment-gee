# Temas a Investigar o Resolver

## Alta prioridad

- [ ] **¿Qué tecnología usar para implementar?** Plugin para OpenCode, plugin para Claude, aplicación independiente, o combo. Decidir cuando pasemos a implementación.

- [ ] **¿Cómo manejar la autenticación de Jira?** Necesitaremos tokens de acceso para el adaptador de salida. Identificar el equipo responsable.

- [ ] **Tratamiento de datos sensibles**: El usuario indicó que para datos de cliente debería usarse Gemini o Claude. Definir política de qué IA puede procesar qué según la sensibilidad de los datos.

## Media prioridad

- [ ] **Integración con Confluence**: El adaptador de salida a Confluence requiere entender su API de contenido.

- [ ] **Formato de exportación a PDF**: Evaluar herramientas de markdown-a-PDF.

- [ ] **Capacity Planner (Proyecto 2)**: La herramienta de Excel-Jira para estimación de velocidad del equipo. Queda pendiente para después del MVP.

## Baja prioridad

- [ ] **Soporte multi-idioma**: Los artefactos deberían poder generarse en español o inglés según el cliente.

- [ ] **Historias de usuario con formato alternativo**: Además del estándar "Como... quiero... para...", podrían necesitarse otros formatos (ej: especificación por casos de uso).
